Mini Search Engine using DSA in C
A mini search engine project built using **C programming** that implements an **Inverted Index** with **Hash Tables and Linked Lists** to efficiently retrieve documents based on keyword searches.  
The backend is integrated with a web-based UI using **CGI**, and the interface is designed with modern aesthetic styling.

Features
-Inverted Index implementation
-Hash Table for fast word lookup
-Linked List to store document occurrences
-Supports multiple documents dynamically
-Keyword frequency ranking
-Web-based UI using HTML + CSS
-Animated and aesthetic frontend
-Result summary and top-ranked document badge

Data Structures Used
1. Hash Table  
2. Linked List  
3. Arrays  
4. File Handling  

Technologies Used

Layer                Technology
Language             C
Data Structures      Hash Table, Linked List
Backend              CGI (Common Gateway Interface) 
FrontendHT           ML, CSS
